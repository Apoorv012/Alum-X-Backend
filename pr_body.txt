feat: add dev-only user creation api

Implements a secured, dev-only endpoint for creating users to facilitate testing and local development.

### Changes
- **Infrastructure**: Added `spring-boot-starter-security` and `UserRepository`.
- **Schema**: Added `username` field to `User` entity.
- **API**: Implemented `DevUserController` (`POST /api/dev/users`).
- **Security**: Added `SecurityConfig` to enable BCrypt and permit dev endpoints.

### Verification
- **Header Check**: Validates `X-DUMMY-TOKEN`.
- **Validation**: Enforces unique username/email and required fields.
- **Hashing**: Passwords are securely hashed before storage.
